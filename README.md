# 🎸 UltimateTab (Sonrise Edition)

A high-performance, distraction-free app for reading guitar chords and tabs during live performances.

## 🍎 For Mac Users: How to Install

1.  **Download**: Get the latest `.dmg` file from the **Actions** or **Releases** tab of this repository.
2.  **Install**: Double-click the downloaded `.dmg` file.
3.  **Drag-and-Drop**: Drag the **UltimateTab** icon into your **Applications** folder.
4.  **Open**: 
    - Go to your Applications folder and open **UltimateTab**.
    - *Note: Since this app is not "notarized" by Apple yet, you might see a warning. If so, right-click (or Control-click) the app icon and select **Open** from the menu, then click **Open** again in the popup.*

## ✨ Highlights

-   **Live Mode**: A dedicated performance view that maximizes song space.
-   **Smart Transpose**: Change keys instantly while keeping your fingerings consistent.
-   **Autoscroll**: Hands-free performance with adjustable speeds.
-   **Offline Support**: Once a song is loaded, it's ready for the stage even without internet.
